# SaneUnits

**A unit conversion and reality-estimation tool that doesn't BS you.**

SaneUnits explains the WHY behind unit confusion — why a 4TB drive shows as 3.64 TiB, why 1000Mbps isn't 1000 MB/s download speed, and how much electricity a 30W device actually costs per year.

## Features

- **Storage**: Decimal vs binary units (TB vs TiB, GB vs GiB)
- **Network**: Bandwidth × file size → real-world transfer time
- **Video**: Bitrate, duration, file size — solve for any one
- **Power**: Watts × usage × electricity rate → annual cost
- **Bilingual**: Chinese / English
- **Light & Dark themes**: Catppuccin Latte & Frappe

## Philosophy

- Accuracy over convenience
- Local-first — all computation in your browser
- Shareable — every calculator state is encoded in URL

## Tech Stack

- React 19 + Vite 6
- Plain CSS (Catppuccin theme)
- No backend, no accounts, no tracking
